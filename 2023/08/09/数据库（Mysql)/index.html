数据库（Mysql)

1.注释

–加空格 或#（单行注释）

2.列出所有数据库

show databases;

3.查看某一个数据库里所有的表

use + 数据库（进入数据库）

show tables;

或者直接 show tables from +数据库名；

4.select

查看当前时间

select now();

select 1 + 1;

查看当前选择的库

select databases();

查看版本

select version();

查看当前数据库用户

select user();

查看数据路径

select @@datadir;

查看mysql安装路径

select @@basedir;

查看mysql安装的系统

select @@version_compile_os;

查询数据

select 查询关键字

select * from mysql.user;

5.增，删，改

创建库

create database test charset utf8mb4;

创建，删除表

creat table t1(id int);

drop table t1;

查看表内填写数据类型：desc 表名

插入数据

insert into t1 values (1,”张三“)，（2，”李四“）（3，”王五“）
